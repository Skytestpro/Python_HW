ID 1. Метод list_to_string класса StringUtils принимает строковый тип данных на входе при выполнении негативного сценария 
Приоритет: Medium       Серьезность: S3 

Описание метода: 
Преобразует список элементов в строку с указанным разделителем
Пример 1: `list_to_string([1,2,3,4]) -> "1, 2, 3, 4"`
Пример 2: `list_to_string(["Sky", "Pro"]) -> "Sky, Pro"`
Пример 3: `list_to_string(["Sky", "Pro"], "-") -> "Sky-Pro"`

Предварительные шаги: 
1. сохранить код класса StringUtils на компьютер
2. открыть VSCode в папке с ранее сохраненным файлом
3. создать новый файл с расширением .py  

Шаги: 
1. Импортировать класс StringUtils в рабочий файл (from /название файла, где лежит класс/ import StringUtils)
2. Обратиться к методу list_to_string со строковыми данными на входе:
res = StringUtils()
print(res.list_to_string('1,2,3,4', '--'))
print(res.list_to_string('1,2,3,4'))

ОР: Появляется ошибка о том, что данные нужно передать в виде списка элементов
ФР: Ошибки не возникает. В консоль выводится результат:
1--,--2--,--3--,--4
1, ,, 2, ,, 3, ,, 4
